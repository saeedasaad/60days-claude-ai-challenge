##  What I Did

For Day 8, I explored Claude Artifacts and learned how AI can generate complete interactive applications instead of simple text outputs.

I used the Environmental Health Analyzer prompt and allowed Claude to build a dashboard application.

The generated application included:

* Interactive Charts
* Data Visualizations
* Dashboard Cards
* Filtering Functionality
* User-Friendly Interface

Claude generated a complete HTML application that could be downloaded and run locally in a browser.
